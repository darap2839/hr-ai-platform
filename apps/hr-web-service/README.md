# JobMatch AI Frontend

Улучшенный фронтенд для платформы подбора кандидатов JobMatch AI.

## Структура проекта

```
frontend/
├── src/
│   ├── components/        # Переиспользуемые компоненты
│   │   ├── Cards/        # Карточки (вакансии, кандидаты)
│   │   ├── Layout/       # Компоненты макета (Header, Container)
│   │   └── UI/           # UI компоненты (Button, Alert, LoadingSpinner)
│   ├── pages/            # Страницы приложения
│   ├── services/         # API сервисы
│   ├── hooks/            # Кастомные React хуки
│   ├── utils/            # Утилиты
│   ├── constants/        # Константы
│   ├── App.jsx           # Главный компонент приложения
│   └── main.jsx          # Точка входа
├── package.json
└── vite.config.js
```

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка

```bash
npm run build
```

## Связь с бэкендом

Фронтенд настроен на работу с бэкендом из проекта `hr-ai-platform`, который должен быть запущен на `http://localhost:8000`.

API эндпоинты:
- `POST /api/v1/upload_vacancy` - загрузка вакансии
- `GET /api/v1/matches` - получение списка кандидатов

## Особенности

- ✅ Модульная архитектура с разделением на сервисы, хуки и утилиты
- ✅ Переиспользуемые UI компоненты

- ✅ Анимации с помощью Framer Motion

- ✅ Типизированные API клиенты
