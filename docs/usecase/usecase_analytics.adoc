= Use Case: Аналитика и отчёты
:toc:
:sectnums:

== Акторы

* hr_analytics_service
* hr_psql_service
* HR-менеджер

== Цель

Предоставить отчёты:
- рэнкинги кандидатов, - статистика вакансий, - воронка найма, - метрики ML.

== Основной сценарий

1. HR пользователь открывает раздел аналитики.
2. hr_api_service отправляет запрос в hr_analytics_service.
3. hr_analytics_service делает запросы в БД.
4. Формирует отчёт (JSON/CSV/PDF).
5. Возвращает данные пользователю.

== Ошибки

* База недоступна → ошибка 503.
* Неверные фильтры → ошибка 400.
