= Use Case: Рекомендации навыков
:toc:
:sectnums:

== Акторы

* hr_matching_service
* hr_ontology_service
* HR пользователь
* hr_api_service

== Цель

Рекомендовать недостающие навыки кандидату или работодателю.

== Основной сценарий

1. HR вызывает API рекомендаций.
2. hr_api_service направляет запрос в hr_matching_service.
3. hr_matching_service анализирует профиль кандидата.
4. Использует Skill Ontology → вычисляет пропуски.
5. Возвращает список навыков и вероятности.

== Альтернативы

* Для вакансии отсутствуют требования → возвращается пустой список.
