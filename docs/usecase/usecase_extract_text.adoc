= Use Case: Извлечение текста из резюме
:toc:
:sectnums:

== Акторы

* Соискатель
* hr_etl_service
* hr_api_service
* OCR / PDF parser
* hr_psql_service

== Цель

Получить чистый текст резюме из файла (PDF, DOCX, JPG).

== Основной сценарий

1. Пользователь загружает файл резюме через Frontend.
2. hr_api_service принимает файл и отправляет его в hr_etl_service.
3. hr_etl_service определяет формат файла.
4. Если требуется OCR — вызывает OCR-пайплайн.
5. Выполняется извлечение текста.
6. hr_etl_service сохраняет сырые и очищенные данные в базе.
7. hr_api_service возвращает ID резюме.

== Ошибки

* Файл не читается → ошибка 422.
* OCR сервис недоступен → ошибка 503.
* Формат не поддерживается → ошибка 400.
